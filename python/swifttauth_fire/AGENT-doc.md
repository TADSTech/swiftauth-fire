# SwifttAuth-Fire (Python/PIP)

This is the backend companion for `swifttauth-fire`. It verifies the Google Auth JWT tokens generated by your frontend and protects your Django or FastAPI routes.

## Installation

```bash
pip install swifttauth-fire firebase-admin
```

## CRITICAL: Firebase Service Account Setup

For your backend to securely verify tokens, it MUST have access to your Firebase Admin Service Account JSON file. 

### How to get the JSON file:
1. Go to your Firebase Console.
2. Navigate to Project Settings > Service accounts.
3. Click Generate new private key.
4. Download the JSON file. 

### Where to put it:
Store this JSON file securely in your backend project folder. DO NOT commit this file to GitHub/Git! (Add `*.json` or the specific filename to your `.gitignore`).

### Setting the Environment Variable:
The library automatically looks for an environment variable named `FIREBASE_SERVICE_ACCOUNT_PATH`.
You MUST set this in your `.env` file or hosting environment:

```env
FIREBASE_SERVICE_ACCOUNT_PATH=/absolute/or/relative/path/to/your/firebase-adminsdk.json
```

*(Alternatively, you can initialize it manually in your app startup: `from swifttauth_fire import init_firebase; init_firebase("path/to/key.json")`)*

## Usage

### 1. Django Setup
Simply use the `@swiftt_auth_required` decorator on your views. 
The client must send the JWT token in the `Authorization: Bearer <TOKEN>` header.

```python
from django.http import JsonResponse
from swifttauth_fire.django import swiftt_auth_required

@swiftt_auth_required
def protected_dashboard(request):
    # If the token is valid, the decorator injects `firebase_user` into the request!
    user = request.firebase_user
    
    return JsonResponse({
        "message": f"Welcome back, {user.get('name')}!",
        "email": user.get('email'),
        "uid": user.get('uid')
    })
```

### 2. FastAPI Setup
Use the `get_current_user` dependency in your route definitions.

```python
from fastapi import FastAPI, Depends
from swifttauth_fire.fastapi import get_current_user

app = FastAPI()

@app.get("/api/protected")
def protected_route(user: dict = Depends(get_current_user)):
    return {
        "message": "You have accessed a protected route!",
        "user_email": user.get("email"),
        "user_id": user.get("uid")
    }
```
